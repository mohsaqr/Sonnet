plot_permutation()

Permutation Test Visualization for Network Comparison

perm <- tna::permutation_test(model1, model2, iter = 1000)
plot_permutation(perm) # shows significant edges only
show_nonsig = FALSE (default)

Significant Only (Default)

Shows only edges that passed the significance test (p < 0.05). Solid colored (green=positive, red=negative) with bold labels and stars.

show_effect = TRUE

With Effect Sizes

Shows effect size in parentheses for significant edges. Absolute effect size values are displayed.

show_nonsig = TRUE

All Differences

Shows all edge differences. Significant: solid colored, bold. Non-significant: dashed grey with lower opacity.

Edge Styling Legend

Significant Positive (p < 0.05, x > y): Solid green (#009900), bold labels with stars
Significant Negative (p < 0.05, x < y): Solid red (#C62828), bold labels with stars
Non-significant: Dashed grey (#888888), plain labels, 40% opacity
