Bootstrap Visualization for TNA Models

boot <- tna::bootstrap(model, iter = 1000)
splot(boot, display = "styled") # default
display = "styled"

Styled (Default)

Significant edges shown as solid dark blue with bold labels. Non-significant edges shown as dashed pink with plain labels. Stars indicate significance level (*, **, ***).

display = "significant"

Significant Only

Shows only edges that passed the significance test (p < 0.05). Non-significant edges are completely removed from the visualization.

display = "full"

Full Network

Shows all edges with uniform styling, ignoring significance. Useful for comparing the full network structure without significance filtering.

display = "ci"

Confidence Intervals

Shows edge weights with confidence interval bounds [lower, upper]. CI bands are visualized as underlays proportional to interval width.

Edge Styling Legend

Significant (p < 0.05): Solid dark blue (#003355), bold labels, rendered on top
Non-significant: Dashed pink (#E091AA), plain labels, rendered behind
